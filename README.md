[![Build Status](https://travis-ci.org/SBR2015/OoTalk_frontend.svg?branch=master)](https://travis-ci.org/SBR2015/OoTalk_frontend)

##Using

```shell
  $ git clone git@github.com:SBR2015/OoTalk_frontend.git
  $ cd OoTalk_frontend
  $ npm install
  $ npm start
  $ gulp
```
